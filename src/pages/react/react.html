Criar um novo projeto em React

O 'react-react-app' é um espaço comfortável para aprender React, e é o melhor jeito de começar a construir uma nova 'single-page-application'(aplicação de uma só página) em React. Ele cria o seu espaço de desenvolvimento para que possa usar as mais modernas 'features', provendo uma boa experiência de desenvolvimento, e otimiza sua aplicação para produção. Para instalar você precisar ter o Node.js >= 6 e npm >= 5.2. Para criar um projeto, dê os seguintes comandos:

npx create-react-app nome-do-seu-app
cd nome-do-seu-app
npm start

Obs.: 'npx' escrito na primeira linha, não é um erro - é uma ferramenta que acelera processos dos pacotes, ela vem na versão 5.2 + do npm.

  • Componentização

    • A componentização é um processo muito importante na programação, criar pequenos blocos de código reutilizáveis. Diminuindo sua complexidade e permitindo que cada componente seja focado em apenas uma funcionalidade ou em um conjunto de funcionalidades semelhantes.

    Dito isso, vamos a primeira parte do projeto Calculadora em React.


    Olhando para uma calculadora, podemos pensar de cara em seus 3 componentes: botões, corpo e tela(onde aparecem os cálculos).

  • Componente Botão

    • Ele vai receber as props do componente App

  • Componente Corpo da Calculadora

    • Este componente vai envolver todos os botões e o componente Display. Que serão passados no App como props.children

  • Componente Display

    • O Display vai receber o state.operations (que é uma Array) como props no componente App.

  • O Componente App

    • Este componente receberá os importes dos outros componentes citados e, além disso, ele receberá a math.js' para lidar com as operações matematicas e a 'immutability-helper' com a função update para ajudar a lidar com as atualizações de state.


    No construtor do App criamos o state operations, que será uma array vazia e será onde colocaremos todas as operações.

    Em seguida temos a função 'calculateOperations', que como o nome diz, vai fazer todos os cáculos, nela usamos a bibliote mathjs com a função eval, que irá fazer as operações matemáticas e a função format pra formatar o numero.


    Teremos aqui mais uma função, a handleClick, que vai lidar com os cliques do usuário. Na constante value recebemos o valor do 'data-value' do botão clicado, e dependendo deste valor realizaremos as funções que ficaram definidas num bloco de switch.


    Finalmente renderizamos o nosso App, jutando todos os Componentes que criamos anteriormente e declarando todas a props necessárias.


    Importante não esquecer de importar o App no index.js e coloca-lo na função 'ReactDOM.render' para possamos ver o projeto funcionando.
